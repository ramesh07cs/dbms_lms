# LMS Frontend

React + Vite frontend for Library Management System (Tribhuvan University, Thapathali Campus).

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Starts at http://localhost:5173. API requests are proxied to the Flask backend (default http://localhost:5000). Set `VITE_API_URL` in `.env` if the backend runs elsewhere.

## Build

```bash
npm run build
```
